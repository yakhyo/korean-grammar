KIIP 1-bosqich — Grammatika va lugʻat

한국어와 한국문화 초급 1 darsligi uchun mustaqil oʻrganish qoʻllanmasi (18 boʻlimli nashr) · Koreya integratsiya dasturi (KIIP)

Ushbu qoʻllanma KIIP 1-bosqich darsligiga (한국어와 한국문화 초급 1) asoslanadi — bu Ijtimoiy integratsiya dasturining birinchi bosqichi boʻlib, koreys alifbosidan boshlovchilar uchun moʻljallangan. Qoʻllanma darslikdagi 18 boʻlim tartibida tuzilgan. Har bir boʻlim oʻz kundalik mavzusini, oʻsha boʻlimda oʻrgatiladigan ikkita grammatik qolipni (shakli, maʼnosi va misol gaplari bilan) hamda tegishli lugʻatni qamrab oladi. Kitob oxirida grammatika koʻrsatkichi keltirilgan.

1-BOʻLIM인사와 소개Salomlashuv va tanishuv

N + 이에요 / 예요 — "...dir (muloyim)"

Shakli undosh bilan tugagan N + 이에요 · unli bilan tugagan N + 예요

Muloyim nutqdagi asosiy "A — B" bogʻlamasi. Otga qoʻshilib, kishining kimligi, kasbi yoki fuqaroligini bildiradi. Tanlash faqat 받침 ga bogʻliq: undosh bilan tugagan ot + 이에요, unli bilan tugagan ot + 예요.

저는 학생이에요.

Men talabaman. (학생 undosh bilan tugaydi → 이에요)

이분은 의사예요.

Bu kishi shifokor. (의사 unli bilan tugaydi → 예요)

제 친구는 중국 사람이에요.

Doʻstim xitoylik.

여기는 학교예요.

Bu yer maktab.

Istisno Biror narsa B emasligini aytish uchun 이에요/예요 ni toʻgʻridan-toʻgʻri inkor qilmaydi — buning oʻrniga alohida inkor bogʻlamasidan foydalaniladi: N + 이/가 아니에요: 저는 학생이에요 → 저는 학생이 아니에요 ("Men talaba emasman").

N + 은 / 는 — "mavzu koʻrsatkichi (...ga kelsak)"

Shakli undosh bilan tugagan N + 은 · unli bilan tugagan N + 는

Gapning mavzusini — nima haqida gapirilayotganini bildiradi. Koʻpincha biror narsani tanishtiradi yoki qarama-qarshi qoʻyadi. Undosh bilan tugagan ot + 은, unli bilan tugagan ot + 는.

제 이름은 마이클이에요.

Mening ismim Maykl. (이름 ㅁ bilan tugaydi → 은)

저는 베트남 사람이에요.

Men vyetnamlikman. (저 unli bilan tugaydi → 는)

저는 커피를 안 마셔요. 그런데 차는 마셔요.

Men qahva ichmayman. Choyni esa ichaman. (qarama-qarshilik)

오늘은 일요일이에요.

Bugun yakshanba.

Qiyoslang 은/는 mavzuni belgilaydi (gap nima haqidaligi, koʻpincha eski yoki qarama-qarshi qoʻyilgan maʼlumot); 이/가 esa egani belgilaydi (koʻpincha yangi maʼlumot yoki "kim/nima?" degan savolning javobi). Keyingi qolipga qarang.

2-BOʻLIM사물Kundalik buyumlar

N + 이 / 가 — "ega koʻrsatkichi"

Shakli undosh bilan tugagan N + 이 · unli bilan tugagan N + 가

Gapning egasini — mavjud boʻlgan, roʻy bergan yoki taʼriflanayotgan narsani bildiradi. Undosh bilan tugagan ot + 이, unli bilan tugagan ot + 가.

책상이 있어요.

Bu yerda parta bor. (책상 undosh bilan tugaydi → 이)

의자가 없어요.

Stul yoʻq. (의자 unli bilan tugaydi → 가)

시계가 비싸요.

Soat qimmat.

가방이 작아요.

Sumka kichkina.

Qiyoslang 이/가 egani belgilaydi (koʻpincha yangi maʼlumot yoki "kim/nima?" degan savolning javobi); 은/는 esa mavzuni belgilaydi. "누가 왔어요? — 친구가 왔어요" javobida yangi ega 가 oladi, 는 emas.

N + 에 있다 / 없다 — "(bir joyda) bor / yoʻq"

Shakli joy N + 에 + 있다 (bor) / 없다 (yoʻq)

Biror narsaning qaysidir joyda borligini yoki yoʻqligini bildiradi. 에 joyni koʻrsatadi; 있다 = "bor", 없다 = "yoʻq".

방에 책상이 있어요.

Xonada parta bor.

교실에 시계가 없어요.

Sinfxonada soat yoʻq.

집에 냉장고가 있어요.

Uyda muzlatgich bor.

가방에 책이 없어요.

Sumkada kitob yoʻq.

Maslahat Joy 에 qoʻshimchasini oladi, ammo mavjud boʻlgan narsa ega koʻrsatkichi 이/가 ni oladi — "방에 책상이 있어요", "방을 책상이…" emas.

N + 위치어 + 에 있다 — "joy / oʻrin soʻzlari"

Shakli N + oʻrin soʻzi + 에 있다/없다 (위, 아래, 앞, 뒤, 옆 …)

Biror narsa qayerda ekanini aytish uchun oʻrin soʻzini tayanch otdan keyin qoʻyib, soʻng 에 있어요 ni qoʻshasiz. Qolip N + 위에 있어요 ("N ning ustida") shaklida boʻladi.

책상 위에 책이 있어요.

Stol ustida kitob bor.

은행은 병원 옆에 있어요.

Bank shifoxonaning yonida.

고양이가 의자 아래에 있어요.

Mushuk stulning ostida.

Qiyoslang 아래 va 밑 ikkalasi ham "ostida" maʼnosini beradi; 밑 koʻproq bevosita tegib turgan, tagidagi narsa haqida (책상 밑), 아래 esa umumiyroq "pastda" maʼnosida. Ikkalasi ham 에 ni oladi: 아래에 / 밑에.

3-BOʻLIM일상생활Kundalik mashgʻulotlar

V + 아 / 어요 — "muloyim hozirgi zamon"

Shakli ㅏ/ㅗ oʻzak + 아요 · boshqa oʻzaklar + 어요 · 하다 → 해요

Hozirgi zamon uchun kundalik muloyim feʼl qoʻshimchasi (yaqin kelajak va yumshoq buyruq maʼnosida ham qoʻllanadi). Oxirgi unlisi ㅏ/ㅗ boʻlgan oʻzaklar 아요 ni, qolganlari 어요 ni oladi, 하다 esa → 해요.

한국어를 배워요.

Men koreys tilini oʻrganaman. (배우다, oʻzak 배우- + 어요 → 배워요)

책을 읽어요.

Men kitob oʻqiyman. (읽- oxirgi unlisi ㅣ → 어요)

물을 마셔요.

Men suv ichaman. (마시다 → 마시 + 어요 → 마셔요)

아침에 운동해요.

Ertalab mashq qilaman. (운동하다 → 해요)

으 탈락 — "ㅡ tushadigan feʼllar (notoʻgʻri)"

Shakli oʻzak oxiridagi ㅡ tushiriladi, soʻng ㅡ dan OLDINGI unliga qarab 아요/어요 qoʻshiladi

Feʼl/sifat oʻzagi ㅡ bilan tugaganda, 아/어요 dan oldin ㅡ tushadi. Qoʻshimcha ㅡ dan oldingi boʻgʻin unlisiga qarab tanlanadi: ㅏ/ㅗ → 아요, aks holda → 어요. Bir boʻgʻinli oʻzaklar (쓰다·크다) da oldida hech narsa yoʻq, shuning uchun standart holatda 어요 boʻladi.

바쁘다 바빠요 아프다 아파요
예쁘다 예뻐요 슬프다 슬퍼요
기쁘다 기뻐요 모으다 모아요
쓰다 써요 끄다 꺼요
크다 커요 — —

오늘 너무 바빠요.

Bugun juda bandman. (바쁘다, 바- da ㅏ bor → 바빠요)

머리가 아파요.

Boshim ogʻriyapti. (아프다, 아- da ㅏ bor → 아파요)

편지를 써요.

Xat yozyapman. (쓰다, bir boʻgʻin → 써요)

Istisno 따르다·들르다·치르다 으 tushadigan feʼllardir (따라요·들러요·치러요). Ammo 모르다·부르다 kabi koʻpchilik boshqa 르-oʻzaklar bunday EMAS — ular 르 notoʻgʻri feʼllaridir (모르다 → 몰라요, 모라요 emas) va 2-bosqichda oʻrgatiladi.

N + 을 / 를 — "toʻldiruvchi koʻrsatkichi"

Shakli undosh bilan tugagan N + 을 · unli bilan tugagan N + 를

Vositasiz toʻldiruvchini — harakat yoʻnaltirilgan narsani bildiradi. Undosh bilan tugagan ot + 을, unli bilan tugagan ot + 를.

음악을 들어요.

Men musiqa tinglayman. (음악 ㄱ bilan tugaydi → 을)

친구를 만나요.

Men doʻstim bilan uchrashaman. (친구 unli bilan tugaydi → 를)

빵을 먹어요.

Men non yeyman.

한국어를 공부해요.

Men koreys tilini oʻrganaman.

4-BOʻLIM장소Joylar va yoʻnalishlar

N + 에 가다 / 오다 — "(bir joyga) bormoq / kelmoq"

Shakli joy N + 에 + 가다 (bormoq) / 오다 (kelmoq)

에 harakatning manzilini bildiradi. Borish uchun 가다, kelish uchun 오다 bilan ishlatiladi.

식당에 가요.

Men oshxonaga boraman.

친구가 한국에 와요.

Doʻstim Koreyaga keladi.

아침에 회사에 가요.

Ertalab ishga boraman.

엄마가 집에 와요.

Onam uyga keladi.

Qiyoslang Manzil 에 (가다/오다) va harakat joyini bildiruvchi 에서: 도서관에 가요 ("kutubxonaga boraman"), ammo 도서관에서 공부해요 ("kutubxonada oʻqiyman"). Keyingi qolipga qarang.

N + 에서 — "...da (harakat sodir boʻladigan joy)"

Shakli joy N + 에서 + harakat feʼli

집에서 밥을 먹어요.

Uyda ovqatlanaman.

Kafeda qahva ichaman.

시장에서 과일을 사요.

Bozordan meva sotib olaman.

5-BOʻLIM날짜와 요일Sana va hafta kunlari

N + 에 — "...da (vaqt)"

Shakli vaqt N + 에 (오늘, 어제, 내일, 지금 bilan ishlatilmaydi)

Biror narsa sodir boʻladigan vaqtni — sana, kun, oy yoki soatni bildiradi. Vaqt soʻziga 에 qoʻshiladi.

월요일에 학교에 가요.

Dushanba kuni maktabga boraman.

3시에 만나요.

Soat 3 da uchrashamiz.

토요일에 친구를 만나요.

Shanba kuni doʻstim bilan uchrashaman.

아침 7시에 일어나요.

Ertalab soat 7 da uygʻonaman.

Istisno 오늘, 어제, 내일, 지금 bilan 에 ishlatilmaydi — 내일 만나요 ("ertaga koʻrishamiz") deyiladi, 내일에× emas.

N + 이 / 가 아니다 — "...emas"

Shakli undosh bilan tugagan N + 이 아니다 · unli bilan tugagan N + 가 아니다

이다 ning inkori — "A — B emas" deyish uchun ishlatiladi. Undosh bilan tugagan ot + 이 아니에요, unli bilan tugagan ot + 가 아니에요.

오늘은 월요일이 아니에요.

Bugun dushanba emas. (월요일 ㄹ bilan tugaydi → 이 아니에요)

저는 학생이 아니에요.

Men talaba emasman.

이것은 제 가방이 아니에요.

Bu mening sumkam emas.

여기는 학교가 아니에요.

Bu yer maktab emas. (학교 unli bilan tugaydi → 가 아니에요)

N + (으)로 — "... tomonga (yoʻnalish)"

Shakli undosh bilan tugagan N + 으로 · unli / ㄹ bilan tugagan N + 로

Bu yerda (으)로 yoʻnalishni — qaysi tomonga yurish yoki burilishni bildiradi. Undosh bilan tugagan N + 으로, unli yoki ㄹ bilan tugagan N + 로.

이쪽으로 오세요.

Marhamat, bu tomonga keling. (이쪽 ㄱ bilan tugaydi → 으로)

오른쪽으로 가세요.

Oʻng tomonga yuring.

2층으로 올라가세요.

Ikkinchi qavatga chiqing.

출구는 저쪽으로 가세요.

Chiqish uchun u tomonga yuring.

Qiyoslang Yoʻnalish 으로 = "...tomonga / ...yoʻnalishida" (학교로 가요, oʻsha tomonga ketmoq); manzil 에 = "...ga yetib bormoq" (학교에 가요, maktabning oʻziga bormoq).

V + 지 마세요 — "...mang (taqiq)"

Shakli feʼl oʻzagi + 지 마세요 (oddiy: 지 마)

Muloyim inkor buyruq — birovga biror ishni qilmaslikni aytadi. Feʼl oʻzagiga 지 마세요 qoʻshiladi (har qanday 받침).

사진을 찍지 마세요.

Surat olmang.

여기에서 담배를 피우지 마세요.

Bu yerda chekmang.

교실에서 음식을 먹지 마세요.

Sinfxonada ovqat yemang.

여기에 쓰레기를 버리지 마세요.

Bu yerga axlat tashlamang.

18-BOʻLIM한국 생활Koreyadagi hayot

A/V + 지요? — "...-a? (tasdiqlatish)"

Shakli feʼl / sifat oʻzagi + 지요? (ogʻzaki: 죠?)

Soʻzlovchi allaqachon toʻgʻri deb bilgan narsani suhbatdoshdan tasdiqlashini soʻraydi. Oʻzakka 지요 qoʻshiladi; nutqda 죠 ga qisqaradi.

한국 음식이 맛있지요?

Koreys taomi mazali-a?

날씨가 춥지요?

Havo sovuq-a?

이 영화 재미있지요?

Bu kino qiziqarli-a?

내일 학교에 가지요?

Ertaga maktabga borasiz-a?

Maslahat 지요? rozilik kutadi, shuning uchun javob odatda 네 boʻladi: "춥죠?" — "네, 추워요." Oddiy 아/어요? esa hech qanday taxminsiz neytral ha/yoʻq savoli boʻlib, undan farq qiladi.

A/V + 는데 — "...-yu, lekin (zamin yoki qarama-qarshilik)"

Shakli feʼl + 는데 · sifat + -(으)ㄴ데 · N + 인데

Asosiy gapdan oldin zamin tayyorlaydi yoki qarama-qarshilik kiritadi. Feʼllar 는데 ni; sifatlar -(으)ㄴ데 ni; otlar 인데 ni oladi.

한국 생활은 힘든데 재미있어요.

Koreyadagi hayot ogʻir, lekin qiziqarli. (힘들다 + ㄴ데, ㄹ tushadi → 힘든데)

지금 비가 오는데 우산이 없어요.

Hozir yomgʻir yogʻyapti, lekin soyabonim yoʻq. (feʼl 오다 → 는데)

배가 고픈데 식당이 문을 닫았어요.

Qornim ochib qolgan, lekin oshxona yopilib qolibdi. (sifat 고프다 → ㄴ데)

이 옷이 예쁜데 좀 비싸요.

Bu kiyim chiroyli, lekin biroz qimmat.

Qiyoslang 지만 sof "lekin"; 는데 yumshoqroq — koʻpincha shunchaki zamin tayyorlaydi ("...-yu, shu sababli...") va iltimosga tabiiy kirish boʻladi: 시간이 있는데 같이 갈까요? ("Vaqtim bor, birga boraylikmi?").
