Referensi ini mengikuti buku teks KIIP Tingkat 2 (한국어와 한국문화 초급 2) — tahap pemula lanjut dalam Program Integrasi Sosial. Materi ini disusun mengikuti 18 unit dalam buku. UNIT 1고향 소개Memperkenalkan Kampung Halaman

N + (이)라고 하다 — "disebut / bernama"

Bentuk N berakhiran konsonan + 이라고 하다 · N berakhiran vokal + 라고 하다

Dipakai untuk menyebutkan nama atau sebutan sesuatu, termasuk saat memperkenalkan diri sendiri. Kata benda berakhiran konsonan + 이라고 하다, kata benda berakhiran vokal + 라고 하다.

제 고향은 부산이라고 해요.

Kampung halaman saya bernama Busan. (부산 berakhiran ㄴ → 이라고)

저는 흐엉이라고 해요.

Nama saya Huong.

이 음식은 한국말로 김치찌개라고 해요.

Makanan ini disebut kimchi jjigae dalam bahasa Korea. (김치찌개 berakhiran vokal → 라고)

제 딸 이름은 민지라고 해요.

Nama anak perempuan saya Minji.

A + (으)ㄴ + N — "kata sifat yang menerangkan kata benda"

Bentuk akar sifat berkonsonan + 은 · akar sifat bervokal + ㄴ, sebelum kata benda

Mengubah kata sifat menjadi keterangan yang diletakkan di depan kata benda ("tempat yang indah"). Akar berkonsonan + 은, akar bervokal + ㄴ.

경치가 아름다운 곳이에요.

Ini tempat yang pemandangannya indah. (아름답다 termasuk ㅂ-tak beraturan → 아름다운)

조용한 동네에 살아요.

Saya tinggal di lingkungan yang tenang. (조용하 berakhiran vokal → ㄴ)

우리 고향은 공기가 깨끗한 곳이라서 살기 좋아요.

Kampung halaman saya tempat yang udaranya bersih, jadi nyaman untuk ditinggali.

날씨가 좋은 날에는 공원에서 산책해요.

Saat cuaca cerah, saya berjalan-jalan di taman. (좋 berakhiran konsonan → 은)

Tips Kata sifat 있다/없다 memakai 는, bukan 은: 맛있는 음식, 재미없는 영화. Perhatikan juga akar ㅂ-tak beraturan: 춥다 → 추운 날, 맵다 → 매운 음식.

UNIT 2집안일Mengerjakan Pekerjaan Rumah

V + (으)ㄹ게요 — "saya akan (lakukan) — menawarkan / niat"

Bentuk akar berkonsonan + 을게요 · akar bervokal + ㄹ게요 (hanya orang pertama)

Menyatakan kemauan pembicara atau sebuah janji/tawaran, sebagai respons terhadap situasi atau lawan bicara. Akar berkonsonan + 을게요, akar bervokal + ㄹ게요.

쓰레기는 제가 버릴게요.

Sampahnya biar saya yang buang. (버리 berakhiran vokal → ㄹ게요)

제가 설거지를 할게요.

Biar saya yang cuci piring.

빨래는 제가 갤게요. 청소기는 남편이 돌려요.

Cuciannya biar saya yang lipat. Suami saya yang pakai penyedot debu.

밥 다 먹을게요. 조금만 기다려 주세요.

Saya selesaikan dulu makannya. Tunggu sebentar, ya. (먹 berakhiran ㄱ → 을게요)

Bandingkan -(으)ㄹ게요 adalah janji yang ditujukan kepada lawan bicara (hanya orang pertama); -(으)ㄹ 래요 menyatakan keinginan sendiri atau menanyakan keinginan lawan bicara secara santai. Jangan katakan 가실게요 — kita tidak bisa berjanji atas nama orang lain.

V + (으)ㄴ 다음에 — "setelah melakukan"

Bentuk akar berkonsonan + 은 다음에 · akar bervokal + ㄴ 다음에 (= 후에)

Menyatakan bahwa aksi kedua terjadi setelah aksi pertama selesai. Akar berkonsonan + 은 다음에, akar bervokal + ㄴ 다음에.

청소한 다음에 쉴 거예요.

Setelah bersih-bersih saya akan istirahat. (청소하 berakhiran vokal → ㄴ 다음에)

밥을 먹은 다음에 약을 드세요.

Minumlah obatnya setelah makan. (먹 berakhiran ㄱ → 은 다음에)

손을 씻은 다음에 요리를 시작해요.

Saya mulai memasak setelah cuci tangan.

빨래를 한 다음에 마트에 갈 거예요.

Setelah mencuci baju, saya akan pergi ke supermarket.

Catatan 다음에 dan 후에 bisa saling menggantikan di sini (먹은 다음에 = 먹은 후에). Untuk kata benda, langsung pakai 후에: 식사 후에 "setelah makan".

UNIT 3옷, 신발Mencoba Pakaian

N + (이)나 — "atau"

Bentuk N berakhiran konsonan + 이나 · N berakhiran vokal + 나

Menggabungkan dua kata benda sebagai pilihan ("A atau B"). Kata benda berakhiran konsonan + 이나, kata benda berakhiran vokal + 나. (Untuk kata kerja, pakai -거나.)

치마나 바지를 입어요.

Saya memakai rok atau celana. (치마 berakhiran vokal → 나)

커피나 차를 마셔요.

Saya minum kopi atau teh.

주말에는 운동화나 샌들을 신어요.

Saat akhir pekan saya memakai sepatu kets atau sandal.

점심으로 빵이나 김밥을 먹을까 해요.

Untuk makan siang saya ingin makan roti atau gimbap. (빵 berakhiran konsonan → 이나)

V + 아 / 어 보다 — "mencoba melakukan"

Bentuk akar ㅏ/ㅗ + 아 보다 · akar lainnya + 어 보다

Berarti mencoba melakukan suatu aksi. Akar ㅏ/ㅗ + 아 보다, akar lainnya + 어 보다. Dalam bentuk lampau (아/어 봤다) artinya "pernah melakukan".

이거 한번 입어 보세요.

Coba pakai yang ini. (입 → 어 보다)

한국 음식을 먹어 봤어요.

Saya pernah mencoba makanan Korea. (bentuk lampau 봤다 = pernah melakukan)

이 신발이 편한지 한번 신어 보세요.

Coba pakai sepatu ini untuk melihat apakah nyaman.

작년에 한복을 입어 봤는데 정말 예뻤어요.

Tahun lalu saya pernah mencoba memakai hanbok, dan benar-benar cantik.

UNIT 4전화Menelepon

V + (으)ㄹ 수 있다 / 없다 — "bisa / tidak bisa"

Bentuk akar berkonsonan + 을 수 있다/없다 · akar bervokal + ㄹ 수 있다/없다

Menyatakan kemampuan atau kemungkinan (있다) serta ketidakmampuan atau ketidakmungkinan (없다). Akar berkonsonan + 을 수 있다/없다, akar bervokal + ㄹ 수 있다/없다.

지금 통화할 수 있어요?

Apakah sekarang bisa bicara di telepon? (통화하 berakhiran vokal → ㄹ 수 있다)

한국 노래를 부를 수 있어요.

Saya bisa menyanyikan lagu Korea.

지금 회의 중이라서 전화를 받을 수 없어요.

Sekarang saya sedang rapat, jadi tidak bisa menerima telepon. (받 berakhiran ㄷ → 을 수 없다)

문자로 주소를 보내 줄 수 있어요?

Bisakah kamu mengirimkan alamatnya lewat pesan singkat?

반말 — "bahasa santai (informal)"

Bentuk hilangkan 요 dari bentuk 아/어요 (먹어요 → 먹어 · 가요 → 가)

Bahasa santai yang dipakai di antara teman dekat dan kepada orang yang lebih muda. Dibentuk dengan menghilangkan 요 dari bentuk 아/어요. Gunakan bahasa sopan kepada orang asing dan orang yang lebih tua.

내일 같이 갈래?

Besok mau pergi bareng? (갈래요 → hilangkan 요 → 갈래)

밥 먹었어?

Sudah makan? (먹었어요 → 먹었어)

지금 어디야? 나 좀 늦을 것 같아.

Sekarang kamu di mana? Sepertinya aku akan sedikit terlambat. (N + 이야/야 menggantikan 이에요/예요)

이따가 전화할게. 끊어.

Nanti aku telepon, ya. Dadah.

Catatan Kopula tidak beraturan dalam 반말: 학생이에요 → 학생이야, 친구예요 → 친구야 (bukan 친구여). Untuk pertanyaan, cukup pertahankan akhiran santai dengan nada naik: 어디 가? "Mau ke mana?"

UNIT 5약, 증상Obat & Penyakit

A/V + (으)면 — "jika / ketika"

Bentuk akar berkonsonan + 으면 · akar bervokal + 면

Membentuk sebuah syarat — "jika / ketika X (terjadi), maka Y". Akar berkonsonan + 으면, akar bervokal + 면.

많이 아프면 이 약을 드세요.

Kalau sangat sakit, minumlah obat ini. (아프 berakhiran vokal → 면)

시간이 있으면 같이 가요.

Kalau ada waktu, ayo pergi bersama. (있 berakhiran ㅆ → 으면)

열이 안 내리면 내일 다시 병원에 오세요.

Kalau demamnya tidak turun, datang lagi ke rumah sakit besok.

약을 먹으면 좀 괜찮아질 거예요.

Kalau minum obat, Anda akan merasa sedikit lebih baik. (먹 berakhiran ㄱ → 으면)

V + 아 / 어서 — "karena / lalu"

Bentuk akar ㅏ/ㅗ + 아서 · akar lainnya + 어서 · 하다 → 해서

Menghubungkan sebab dengan akibatnya, atau dua aksi berurutan yang berkaitan erat. Akar ㅏ/ㅗ + 아서, akar lainnya + 어서, 하다 → 해서. Klausa pertama tidak diberi penanda waktu.

감기에 걸려서 병원에 갔어요.

Saya pergi ke rumah sakit karena terkena flu. (걸리 + 어서 → 걸려서)

친구를 만나서 같이 밥을 먹었어요.

Saya bertemu teman lalu kami makan bersama. (urutan aksi, bukan sebab)

머리가 너무 아파서 약국에서 두통약을 샀어요.

Kepala saya sakit sekali, jadi saya membeli obat sakit kepala di apotek.

은행에 가서 통장을 만들었어요.

Saya pergi ke bank lalu membuka rekening.

Bandingkan -아/어서 tidak bisa diikuti perintah atau ajakan — untuk itu gunakan -(으)니까. Jadi 늦어서 미안해요 ("maaf saya terlambat") boleh, tetapi untuk "Ayo cepat karena kita terlambat" Anda harus mengatakan 늦었으니까 서두르세요, bukan 늦어서 서두르세요.

A/V + 겠 — "akan / pasti (niat, dugaan)"

Bentuk akar kata kerja / kata sifat + 겠 + akhiran (-겠어요, -겠습니다)

Menunjukkan niat/kemauan pembicara (orang pertama), atau dugaan tentang suatu keadaan (orang lain). Akar kata kerja/kata sifat + 겠 + akhiran.

내일부터 열심히 하겠어요.

Mulai besok saya akan giat berusaha. (orang pertama → niat)

시험이 어렵겠어요.

Sepertinya ujiannya akan sulit. (dugaan)

Anda pasti sangat senang karena lulus.

이번 일은 제가 책임지고 하겠습니다.

Untuk tugas ini saya akan bertanggung jawab dan mengerjakannya.

A/V + (으)ㄹ 때 — "ketika"

Bentuk akar berkonsonan + 을 때 · akar bervokal + ㄹ 때 (lampau: -았/었을 때)

Menandai waktu atau saat sesuatu terjadi. Akar berkonsonan + 을 때, akar bervokal + ㄹ 때 (lampau: -았/었을 때).

맛있는 음식을 먹을 때 행복해요.

Saya bahagia ketika makan makanan yang enak. (먹 berakhiran ㄱ → 을 때)

어렸을 때 시골에서 살았어요.

Ketika kecil, saya tinggal di desa. (keadaan lampau → -었을 때)

고향 생각이 날 때 가족에게 전화해요.

Ketika rindu kampung halaman, saya menelepon keluarga. (나 berakhiran vokal → ㄹ 때)

기분이 안 좋을 때 음악을 들어요.

Ketika suasana hati sedang buruk, saya mendengarkan musik.

ㄷ 불규칙 — "konjugasi ㄷ-tak beraturan"

Bentuk ㄷ di akhir akar ㄷ → ㄹ sebelum akhiran bervokal (아/어요, 았/었어요, 으면…)

Pada kata kerja ini, ㄷ di akhir akar melunak menjadi ㄹ setiap kali diikuti akhiran bervokal. Sebelum akhiran berkonsonan (-고, -지, -습니다), ㄷ tetap.

Kata kerja -아/어요 -았/었어요 -(으)면
듣다 (mendengar) 들어요 들었어요 들으면
걷다 (berjalan) 걸어요 걸었어요 걸으면
묻다 (bertanya) 물어요 물었어요 물으면
싣다 (memuat) 실어요 실었어요 실으면

음악을 들으면서 산책했어요.

Saya berjalan-jalan sambil mendengarkan musik. (듣 + 으면서 → 들으면서)

길을 몰라서 경찰에게 물어봤어요.

Saya tidak tahu jalan, jadi bertanya kepada polisi. (묻 + 어 → 물어)

매일 30분씩 걸어요.

Saya berjalan kaki 30 menit setiap hari.

Pengecualian Banyak kata kerja berakhiran ㄷ tetap beraturan (mempertahankan ㄷ): 받다 → 받아요, 닫다 → 닫아요, 믿다 → 믿어요. Dan 묻다 yang berarti "mengubur" beraturan → 묻어요, sedangkan 묻다 "bertanya" tak beraturan → 물어요.

ㅂ 불규칙 — "konjugasi ㅂ-tak beraturan"

Bentuk ㅂ di akhir akar ㅂ → 우 sebelum akhiran bervokal (kebanyakan kata sifat)

ㅂ di akhir akar berubah menjadi 우 dan menyatu dengan akhiran: -아/어요 menjadi -워요, -(으)면 menjadi -우면.

Sifat/Kerja -아/어요 -았/었어요 -(으)ㄴ
춥다 (dingin) 추워요 추웠어요 추운
덥다 (panas) 더워요 더웠어요 더운
맵다 (pedas) 매워요 매웠어요 매운
어렵다 (sulit) 어려워요 어려웠어요 어려운
돕다 (menolong) 도와요 도왔어요 도운

오늘은 날씨가 너무 추워요.

Cuaca hari ini sangat dingin. (춥 + 어요 → 추워요)

이 김치찌개가 정말 매워요.

Kimchi jjigae ini benar-benar pedas. (맵 + 어요 → 매워요)

짐이 무거우면 제가 도와 드릴게요.

Kalau barangnya berat, biar saya bantu. (돕 memakai 와, bukan 워 → 도와요)

Pengecualian Hanya 돕다 dan 곱다 yang memakai 와 (도와요, 고와요); yang lain memakai 워. Sebagian kata kerja berakhiran ㅂ tetap beraturan (mempertahankan ㅂ): 입다 → 입어요, 좁다 → 좁아요, 잡다 → 잡아요.

르 불규칙 — "konjugasi 르-tak beraturan"

Bentuk 르 di akhir akar → ㄹㄹ sebelum 아/어 (muncul ㄹ tambahan)

Ketika akar berakhiran 르 bertemu 아/어, 으 hilang dan muncul satu ㄹ tambahan: 모르 + 아 → 몰라, 부르 + 어 → 불러.

Kerja/Sifat -아/어요 -았/었어요
모르다 (tidak tahu) 몰라요 몰랐어요
부르다 (memanggil/menyanyi) 불러요 불렀어요
빠르다 (cepat) 빨라요 빨랐어요
다르다 (berbeda) 달라요 달랐어요
자르다 (memotong) 잘라요 잘랐어요

죄송하지만 저도 잘 몰라요.

Maaf, tetapi saya juga kurang tahu. (모르 + 아요 → 몰라요)

생일에 다 같이 노래를 불렀어요.

Saat ulang tahun kami semua menyanyi bersama. (부르 + 었어요 → 불렀어요)

제 생각은 좀 달라요.

Pendapat saya sedikit berbeda.

Pengecualian 따르다 (mengikuti/menuang) dan 들르다 (mampir) terlihat mirip, tetapi termasuk 으 탈락, bukan 르-tak beraturan: 따르 + 아 → 따라요, 들르 + 어 → 들러요 (tanpa ㄹ ganda).

ㅎ 불규칙 — "konjugasi ㅎ-tak beraturan (kata sifat)"

Bentuk ㅎ di akhir akar hilang; dengan 아/어 vokalnya menjadi ㅐ (atau ㅒ)

Kata sifat warna dan jenis 그렇다 menghilangkan ㅎ sebelum akhiran. Sebelum 아/어요 hasilnya -애요; bentuk keterangannya -(으)ㄴ dengan ㅎ yang hilang (빨간, 파란).

Kata sifat -아/어요 -(으)ㄴ
그렇다 (begitu) 그래요 그런
빨갛다 (merah) 빨개요 빨간
파랗다 (biru) 파래요 파란
노랗다 (kuning) 노래요 노란
하얗다 (putih) 하얘요 하얀
어떻다 (bagaimana) 어때요 어떤

하늘이 정말 파래요.

Langitnya benar-benar biru. (파랗 → ㅎ hilang, ㅏ+아 → ㅐ → 파래요)

빨간 사과를 두 개 샀어요.

Saya membeli dua apel merah. (bentuk keterangan 빨간, ㅎ hilang)

날씨가 어때요?

Bagaimana cuacanya? (어떻 + 어요 → 어때요)

Pengecualian Kata umum berakhiran ㅎ tetap beraturan (mempertahankan ㅎ): 좋다 → 좋아요, 많다 → 많아요, 넣다 → 넣어요, 놓다 → 놓아요. Hanya kata sifat penggambar jenis 그렇다/색깔 yang tak beraturan.

ㅅ 불규칙 — "konjugasi ㅅ-tak beraturan"

Bentuk ㅅ di akhir akar hilang sebelum akhiran bervokal — tetapi tanpa penyatuan vokal

ㅅ menghilang sebelum akhiran bervokal, tetapi kedua vokal tetap terpisah (tidak menyatu): 짓 + 어요 → 지어요, bukan 져요.

Kerja/Sifat -아/어요 -았/었어요 -(으)면
짓다 (membangun) 지어요 지었어요 지으면
낫다 (sembuh) 나아요 나았어요 나으면
붓다 (menuang/bengkak) 부어요 부었어요 부으면
잇다 (menyambung) 이어요 이었어요 이으면

감기가 빨리 나으면 좋겠어요.

Semoga flu saya cepat sembuh. (낫 + 으면 → 나으면)

목이 부어서 말하기가 힘들어요.

Tenggorokan saya bengkak, jadi sulit berbicara. (붓 + 어서 → 부어서, tanpa penyatuan vokal)

컵에 물을 부어 주세요.

Tolong tuangkan air ke gelas.

Pengecualian Beberapa kata kerja berakhiran ㅅ tetap beraturan (mempertahankan ㅅ): 웃다 → 웃어요, 씻다 → 씻어요, 벗다 → 벗어요.

UNIT 17건강한 생활Menjaga Kesehatan

A/V + (으)면 좋겠다 — "saya harap / berharap…"

Bentuk akar + 으면/면 좋겠다 (juga -았/었으면 좋겠다)

Menyatakan harapan tentang sesuatu yang belum terwujud. Akar + 으면/면 좋겠다; bentuk lampau -았/었으면 좋겠다 membuat harapannya lebih kuat.

잠을 푹 자면 좋겠어요.

Saya berharap bisa tidur nyenyak. (자 berakhiran vokal → 면 좋겠다)

빨리 건강해졌으면 좋겠어요.

Saya berharap segera pulih dan sehat. (-었으면 → harapan lebih kuat)

주말에는 날씨가 맑으면 좋겠어요.

Saya berharap cuacanya cerah saat akhir pekan. (맑 berakhiran ㄺ → 으면)

우리 가족이 모두 건강하면 좋겠어요.

Saya berharap seluruh keluarga saya tetap sehat.

N + 에 — "per (frekuensi / laju)"

Bentuk N waktu/satuan + 에 + angka + penggolong

Di sini 에 menandai laju atau frekuensi — "X kali per (hari / minggu)". Partikel ini menempel pada kata benda waktu/satuan.

하루에 30분씩 운동해요.

Saya berolahraga 30 menit per hari. (per hari)

일주일에 세 번 수업이 있어요.

Ada kelas tiga kali seminggu.

한 달에 한 번 건강 검진을 받아요.

Saya melakukan pemeriksaan kesehatan sekali sebulan.

하루에 물을 여덟 잔씩 마셔요.

Saya minum delapan gelas air per hari.

Catatan Ini partikel 에 yang sama yang dipakai untuk waktu dan tempat, di sini berarti "per". Padukan dengan 씩 ("masing-masing/setiap") untuk "X jumlah per satuan": 하루에 두 번씩.
