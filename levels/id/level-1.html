Tingkat 1 · 초급 1

Referensi belajar mandiri untuk 한국어와 한국문화 초급 1 (edisi 18 unit) · Program Imigrasi & Integrasi Korea

Referensi ini mengikuti buku teks KIIP Tingkat 1 (한국어와 한국문화 초급 1) — tahap pertama Program Integrasi Sosial, untuk pemula yang memulai dari abjad Korea. Materi disusun mengikuti 18 unit dalam buku tersebut. Setiap unit memperkenalkan topik sehari-harinya, dua pola tata bahasa yang diajarkan di dalamnya (lengkap dengan bentuk, makna, dan contoh kalimat), serta kosakata yang sesuai. Sebuah indeks tata bahasa menutup buku ini.

한글 · Abjad Korea

Bahasa Korea ditulis dengan 한글 (Hangul), sebuah abjad yang terdiri dari 14 konsonan dasar dan 10 vokal dasar. Huruf-huruf digabung menjadi blok suku kata (mis. ㅎ + ㅏ + ㄴ → 한).

UNIT 1인사와 소개Sapaan & Perkenalan

N + 이에요 / 예요 — "(adalah), bentuk sopan"

Bentuk N berakhiran konsonan + 이에요 · N berakhiran vokal + 예요

Kopula dasar "A adalah B" dalam ragam sopan. Tempelkan pada kata benda untuk menyatakan identitas, pekerjaan, atau kewarganegaraan. Pilihannya murni berdasarkan 받침: kata benda berakhiran konsonan + 이에요, kata benda berakhiran vokal + 예요.

저는 학생이에요.

Saya seorang pelajar. (학생 berakhiran konsonan → 이에요)

이분은 의사예요.

Orang ini seorang dokter. (의사 berakhiran vokal → 예요)

제 친구는 중국 사람이에요.

Teman saya orang Tiongkok.

여기는 학교예요.

Ini adalah sekolah.

Pengecualian Untuk mengatakan bahwa sesuatu bukan B, jangan menegasikan 이에요/예요 secara langsung — gunakan kopula negatif tersendiri N + 이/가 아니에요: 저는 학생이에요 → 저는 학생이 아니에요 ("Saya bukan pelajar").

N + 은 / 는 — "penanda topik (kalau soal…)"

Bentuk N berakhiran konsonan + 은 · N berakhiran vokal + 는

Menandai topik kalimat — hal yang sedang dibicarakan. Sering dipakai untuk memperkenalkan atau membandingkan. Kata benda berakhiran konsonan + 은, kata benda berakhiran vokal + 는.

제 이름은 마이클이에요.

Nama saya Michael. (이름 berakhiran ㅁ → 은)

저는 베트남 사람이에요.

Saya orang Vietnam. (저 berakhiran vokal → 는)

저는 커피를 안 마셔요. 그런데 차는 마셔요.

Saya tidak minum kopi. Tapi kalau teh, saya minum. (kontras)

오늘은 일요일이에요.

Hari ini hari Minggu.

Bandingkan 은/는 menandai topik (hal yang dibicarakan kalimat, sering informasi lama atau yang dikontraskan); 이/가 menandai subjek (sering informasi baru atau jawaban atas "siapa/apa?"). Lihat pola berikutnya.

UNIT 2사물Benda Sehari-hari

N + 이 / 가 — "penanda subjek"

Bentuk N berakhiran konsonan + 이 · N berakhiran vokal + 가

Menandai subjek gramatikal — hal yang ada, terjadi, atau digambarkan. Kata benda berakhiran konsonan + 이, kata benda berakhiran vokal + 가.

책상이 있어요.

Ada sebuah meja. (책상 berakhiran konsonan → 이)

의자가 없어요.

Tidak ada kursi. (의자 berakhiran vokal → 가)

시계가 비싸요.

Jam itu mahal.

가방이 작아요.

Tas itu kecil.

Bandingkan 이/가 menandai subjek (sering informasi baru atau jawaban atas "siapa/apa?"); 은/는 menandai topik. Dalam jawaban seperti "누가 왔어요? — 친구가 왔어요", subjek baru memakai 가, bukan 는.

N + 에 있다 / 없다 — "berada / tidak berada di"

Bentuk N tempat + 에 + 있다 (ada) / 없다 (tidak ada)

Menyatakan bahwa sesuatu ada atau tidak ada di suatu tempat. 에 menandai lokasi; 있다 = "ada", 없다 = "tidak ada".

방에 책상이 있어요.

Ada meja di dalam kamar.

교실에 시계가 없어요.

Tidak ada jam di ruang kelas.

집에 냉장고가 있어요.

Ada kulkas di rumah.

가방에 책이 없어요.

Tidak ada buku di dalam tas.

Tips Lokasi memakai 에, tetapi benda yang ada memakai penanda subjek 이/가 — "방에 책상이 있어요", bukan "방을 책상이…".

N + 위치어 + 에 있다 — "kata posisi / lokasi"

Bentuk N + kata posisi + 에 있다/없다 (위, 아래, 앞, 뒤, 옆 …)

Untuk menyatakan di mana sesuatu berada, letakkan kata posisi setelah kata benda acuan, lalu tambahkan 에 있어요. Polanya adalah N + 위에 있어요 ("berada di atas N").

Posisi Makna Posisi Makna

앞 di depan 뒤 di belakang
옆 di samping 안 di dalam
밖 di luar 사이 di antara
왼쪽 kiri 오른쪽 kanan
맞은편 di seberang — —

책상 위에 책이 있어요.

UNIT 3일상생활Kegiatan Sehari-hari

V + 아 / 어요 — "bentuk sopan masa kini"

Bentuk kata dasar ㅏ/ㅗ + 아요 · kata dasar lain + 어요 · 하다 → 해요

Akhiran kata kerja sopan sehari-hari untuk masa kini (juga dipakai untuk masa depan yang dekat dan perintah yang halus). Kata dasar dengan vokal terakhir ㅏ/ㅗ memakai 아요, yang lain memakai 어요, dan 하다 → 해요.

한국어를 배워요.

Saya belajar bahasa Korea. (배우다, kata dasar 배우- + 어요 → 배워요)

책을 읽어요.

Saya membaca buku. (읽-, vokal terakhir ㅣ → 어요)

물을 마셔요.

Saya minum air. (마시다 → 마시 + 어요 → 마셔요)

아침에 운동해요.

Saya berolahraga di pagi hari. (운동하다 → 해요)

으 탈락 — "kata kerja luluh ㅡ (tidak beraturan)"

Bentuk luluhkan ㅡ di akhir kata dasar, lalu tambahkan 아요/어요 sesuai vokal SEBELUM ㅡ

Bila kata dasar kata kerja/kata sifat berakhiran ㅡ, ㅡ luluh sebelum 아/어요. Akhirannya ditentukan oleh vokal suku kata sebelum ㅡ: ㅏ/ㅗ → 아요, selain itu → 어요. Kata dasar satu suku kata (쓰다·크다) tidak punya vokal sebelumnya, jadi defaultnya 어요.

Kata kerja -아/어요 Kata kerja -아/어요
바쁘다 바빠요 아프다 아파요
예쁘다 예뻐요 슬프다 슬퍼요
기쁘다 기뻐요 모으다 모아요
쓰다 써요 끄다 꺼요
크다 커요 — —

오늘 너무 바빠요.

Hari ini saya sangat sibuk. (바쁘다, 바- bervokal ㅏ → 바빠요)

머리가 아파요.

Kepala saya sakit. (아프다, 아- bervokal ㅏ → 아파요)

편지를 써요.

Saya menulis surat. (쓰다, satu suku kata → 써요)

Pengecualian 따르다·들르다·치르다 TERMASUK kata kerja luluh 으 (따라요·들러요·치러요). Tetapi sebagian besar kata dasar 르 lain seperti 모르다·부르다 BUKAN — keduanya termasuk 르 불규칙 (모르다 → 몰라요, bukan 모라요), yang diajarkan di Tingkat 2.

N + 을 / 를 — "penanda objek"

Bentuk N berakhiran konsonan + 을 · N berakhiran vokal + 를

Menandai objek langsung — hal yang dikenai tindakan. Kata benda berakhiran konsonan + 을, kata benda berakhiran vokal + 를.

음악을 들어요.

Saya mendengarkan musik. (음악 berakhiran ㄱ → 을)

친구를 만나요.

Saya bertemu teman. (친구 berakhiran vokal → 를)

빵을 먹어요.

Saya makan roti.

한국어를 공부해요.

Saya belajar bahasa Korea.

UNIT 4장소Tempat & Arah

N + 에 가다 / 오다 — "pergi / datang ke (suatu tempat)"

Bentuk N tempat + 에 + 가다 (pergi) / 오다 (datang)

에 menandai tujuan perpindahan. Dipakai dengan 가다 (pergi) dan 오다 (datang).

식당에 가요.

Saya pergi ke restoran.

친구가 한국에 와요.

Teman saya datang ke Korea.

아침에 회사에 가요.

Saya pergi ke kantor di pagi hari.

엄마가 집에 와요.

Ibu pulang ke rumah.

Bandingkan Tujuan 에 (가다/오다) vs. lokasi tindakan 에서: 도서관에 가요 ("pergi ke perpustakaan") tetapi 도서관에서 공부해요 ("belajar di perpustakaan"). Lihat pola berikutnya.

N + 에서 — "di (tempat suatu tindakan terjadi)"

Bentuk N tempat + 에서 + kata kerja tindakan

Menandai lokasi tempat suatu tindakan berlangsung (berbeda dengan 에, yang menandai keberadaan atau tujuan). Tempelkan 에서 pada kata benda tempat.

도서관에서 공부해요.

Saya belajar di perpustakaan.

집에서 밥을 먹어요.

Saya makan di rumah.

카페에서 커피를 마셔요.

Saya minum kopi di kafe.

시장에서 과일을 사요.

Saya membeli buah di pasar.

UNIT 5날짜와 요일Tanggal & Hari

N + 에 — "pada (suatu waktu)"

Bentuk N waktu + 에 (tidak dipakai dengan 오늘, 어제, 내일, 지금)

Menandai waktu terjadinya sesuatu — tanggal, hari, bulan, atau jam. Tempelkan 에 pada kata waktu.

월요일에 학교에 가요.

Saya pergi ke sekolah pada hari Senin.

3시에 만나요.

Mari bertemu pukul 3.

토요일에 친구를 만나요.

Saya bertemu teman pada hari Sabtu.

아침 7시에 일어나요.

Saya bangun pukul 7 pagi.

Pengecualian Jangan gunakan 에 dengan 오늘, 어제, 내일, 지금 — katakan 내일 만나요 ("sampai jumpa besok"), bukan 내일에×.

N + 이 / 가 아니다 — "bukan (sesuatu)"

Bentuk N berakhiran konsonan + 이 아니다 · N berakhiran vokal + 가 아니다

Bentuk negatif dari 이다 — dipakai untuk mengatakan "A bukan B". Kata benda berakhiran konsonan + 이 아니에요, kata benda berakhiran vokal + 가 아니에요.

오늘은 월요일이 아니에요.

Hari ini bukan hari Senin. (월요일 berakhiran ㄹ → 이 아니에요)

저는 학생이 아니에요.

Saya bukan pelajar.

이것은 제 가방이 아니에요.

Ini bukan tas saya.

여기는 학교가 아니에요.

Ini bukan sekolah. (학교 berakhiran vokal → 가 아니에요)

UNIT 6하루 일과Daily Schedule

N부터 N까지 — "from … to …"

Form start N + 부터 · end N + 까지 (time span)

Marks the start (부터) and end (까지) of a span of time. For places, use 에서 … 까지.

9시부터 6시까지 일해요.

I work from 9 to 6.

월요일부터 금요일까지 공부해요.

I study from Monday to Friday.

1시부터 2시까지 점심을 먹어요.

I eat lunch from 1 to 2.

아침부터 저녁까지 바빠요.

I'm busy from morning to evening.

Compare For a span between two places, swap 부터 for 에서: 집에서 회사까지 30분 걸려요 ("It takes 30 minutes from home to the office").

안 + V — "do not (negation)"

Form 안 + verb/adjective  ·  하다-verbs: N + 안 하다

The simple negative — place 안 right before the verb.

아침을 안 먹어요.

I don't eat breakfast.

오늘은 일 안 해요.

I don't work today. (운동하다-type: 안 splits the noun + 하다)

저는 커피를 안 마셔요.

I don't drink coffee.

A polite command or request. Consonant stem + 으세요, vowel stem + 세요. The honorific present is the same form (먹다 → 드세요).

Verb -ㅂ니다 -(으)세요 -는
살다 삽니다 사세요 사는
알다 압니다 아세요 아는
만들다 만듭니다 만드세요 만드는
팔다 팝니다 파세요 파는

Joins two nouns ("A and B"), or means "with (someone)". 하고 attaches the same way regardless of 받침.

The formal-polite style used in announcements, presentations, and business settings. Consonant stem + 습니다, vowel stem + ㅂ니다; questions end in 습니까/ㅂ니까.

Raises respect toward the subject of the sentence (elders, guests). Some verbs are special: 먹다 → 드시다, 자다 → 주무시다.

UNIT 13교통Transportation

N + (으)로 — "by means of (transport / tool)"

Form consonant-final N + 으로 · vowel / ㄹ-final N + 로

Marks the method, means, or instrument — including transport. Consonant-final N + 으로, vowel-final or ㄹ-final N + 로.

버스로 학교에 가요.

I go to school by bus. (버스 ends in a vowel → 로)

젓가락으로 먹어요.

I eat with chopsticks. (젓가락 ends in ㄱ → 으로)

지하철로 회사에 가요.

I go to work by subway. (지하철 ends in ㄹ → 로, not 으로)

볼펜으로 이름을 써요.

I write my name with a pen.

Exception After a ㄹ 받침, use plain 로, not 으로: 지하철로, 연필로 — ㄹ patterns with vowels here.

V + (으)러 가다 / 오다 — "go / come (in order) to do"

Form consonant stem + 으러 · vowel stem + 러 + 가다/오다

States the purpose of going or coming somewhere. Consonant stem + 으러, vowel stem + 러, then 가다/오다.

밥을 먹으러 식당에 가요.

I go to the restaurant to eat. (먹- consonant stem → 으러)

친구를 만나러 왔어요.

I came to meet a friend. (만나- vowel stem → 러)

옷을 사러 백화점에 가요.

I go to the department store to buy clothes.

책을 빌리러 도서관에 가요.

I go to the library to borrow a book.

Tip (으)러 only pairs with motion verbs (가다·오다·다니다). For a general purpose with any verb, use (으)려고 (Level 2).

UNIT 14약속Making Plans

V + (으)ㄹ까요? — "shall we? / shall I?"

Form consonant stem + 을까요? · vowel stem + ㄹ까요?

Suggests doing something together or offers to do something, and asks the listener's opinion. Consonant stem + 을까요, vowel stem + ㄹ까요.

저녁 7시에 만날까요?

Shall we meet at 7 p.m.? (만나- vowel stem → ㄹ까요)

제가 도와드릴까요?

Shall I help you?

우리 같이 점심 먹을까요?

Shall we have lunch together? (먹- consonant stem → 을까요)

내일 영화 볼까요?

Shall we watch a movie tomorrow?

Tip Answer a 을까요? suggestion with -(으)ㅂ시다 ("let's") or 아/어요: "네, 같이 먹읍시다" / "네, 먹어요" ("Yes, let's eat").

못 + V — "cannot (inability)"

Form 못 + verb  ·  하다-verbs: N + 못 하다

Expresses inability or impossibility — couldn't, can't. Place 못 right before the verb (contrast 안 = simply doesn't).

오늘은 못 가요.

I can't go today.

수영을 못 해요.

I can't swim. (하다-verb: 못 splits the noun + 하다)

매운 음식을 못 먹어요.

I can't eat spicy food.

바빠서 친구를 못 만나요.

I'm busy, so I can't meet my friend.

Compare 안 = choose not to (안 먹어요, "I don't eat it"); 못 = unable to (못 먹어요, "I can't eat it"). Like 안, 못 splits 하다-verbs: 운동 못 해요.

UNIT 15날씨Weather & Seasons

A/V + 네요 — "(oh,) … isn't it! (realization)"

Form Verb / Adjective stem + 네요

Expresses mild surprise or a fresh observation about something you've just noticed. Attach 네요 to the stem.

오늘 날씨가 덥네요.

The weather is hot today!

한국어를 잘하네요.

Oh, you speak Korean well!

이 음식이 정말 맛있네요.

Wow, this food is really delicious!

밖에 비가 오네요.

Oh, it's raining outside!

Exception A ㄹ-stem drops ㄹ before 네요: 살다 → 사네요 ("Oh, you live here!"), 멀다 → 머네요 ("It's far, isn't it!").

N + 보다 — "than (comparison)"

Form N + 보다 (+ 더 "more") + adjective

Marks the standard of comparison — "more … than N". 보다 attaches to the thing being compared against, often paired with 더 ("more").

서울이 부산보다 추워요.

Seoul is colder than Busan.

오늘이 어제보다 더 더워요.

Today is hotter than yesterday.

기차가 버스보다 빨라요.

The train is faster than the bus.

동생이 저보다 키가 커요.

My younger sibling is taller than me.

Tip Word order is flexible: 부산보다 서울이 추워요 means the same thing. The noun with 보다 is the baseline you measure against.

UNIT 16병원Health & Hospital

-아 / 어서 — "because / so (reason)"

Form ㅏ/ㅗ stem + 아서 · other stems + 어서 · 하다 → 해서

Links a reason to its result. ㅏ/ㅗ stem + 아서, other stems + 어서, 하다 → 해서. The first clause cannot carry its own tense (tense goes on the final verb).

배가 아파서 병원에 갔어요.

I went to the hospital because my stomach hurt. (no 았 on 아파서)

바빠서 점심을 못 먹었어요.

I was busy, so I couldn't eat lunch.

날씨가 추워서 집에 있어요.

It's cold, so I stay at home.

머리가 아파서 약을 먹어요.

My head hurts, so I take medicine.

Exception 아/어서 can't be used before a command or suggestion — for "because" with 으세요/읍시다, use (으)니까 instead: 추우니까 문을 닫으세요 ("It's cold, so close the door").

V + 는 것 — "the act of …-ing (noun form)"

Form Verb stem + 는 것 (often shortened to 는 게 / 는 걸)

Turns a verb into a noun phrase so it can be a subject or object. Verb stem + 는 것, often shortened to 는 게 (+이) / 는 걸 (+을) in speech.

운동하는 것이 건강에 좋아요.

Exercising is good for your health.

저는 책 읽는 것을 좋아해요.

I like reading books.

저는 노래하는 것을 좋아해요.

I like singing.

아침에 일어나는 것이 힘들어요.

Getting up in the morning is hard.

Tip In casual speech 는 것이 → 는 게 and 는 것을 → 는 걸: 운동하는 게 좋아요, 책 읽는 걸 좋아해요.

UNIT 17공공장소Public Spaces

N + (으)로 — "toward (direction)"

Form consonant-final N + 으로 · vowel / ㄹ-final N + 로

Here (으)로 marks direction — which way to move or turn. Consonant-final N + 으로, vowel- or ㄹ-final N + 로.

이쪽으로 오세요.

Please come this way. (이쪽 ends in ㄱ → 으로)

오른쪽으로 가세요.

Go to the right.

2층으로 올라가세요.

Go up to the second floor.

출구는 저쪽으로 가세요.

For the exit, go that way.

Compare Direction 으로 = "toward / in the direction of" (학교로 가요, heading that way); destination 에 = "arriving at" (학교에 가요, going to the school itself).

V + 지 마세요 — "please don't (prohibition)"

Form Verb stem + 지 마세요 (plain: 지 마)

A polite negative command — telling someone not to do something. Attach 지 마세요 to the verb stem (any 받침).

사진을 찍지 마세요.

Please don't take photos.

여기에서 담배를 피우지 마세요.

Please don't smoke here.

교실에서 음식을 먹지 마세요.

Please don't eat food in the classroom.

여기에 쓰레기를 버리지 마세요.

Please don't throw trash here.

Compare 지 마세요 forbids an action ("don't do it"); the affirmative request is (으)세요 ("please do it"). Casually, 지 마세요 → 지 마: 걱정하지 마 ("don't worry").

UNIT 18한국 생활Life in Korea

A/V + 지요? — "…, right? (confirming)"

Form Verb / Adjective stem + 지요? (spoken: 죠?)

Asks the listener to confirm something you already believe to be true. Attach 지요 to the stem; in speech it contracts to 죠.

한국 음식이 맛있지요?

Korean food is delicious, right?

날씨가 춥지요?

It's cold, isn't it?

이 영화 재미있지요?

This movie is fun, right?

내일 학교에 가지요?

You're going to school tomorrow, right?

Tip 지요? expects agreement, so the answer is usually 네: "춥죠?" — "네, 추워요." Compare plain 아/어요? which is a neutral yes/no question with no expectation.

A/V + 는데 — "but / and (background, contrast)"

Form Verb + 는데 · Adjective + -(으)ㄴ데 · N + 인데

Gives background or sets up a contrast before the main clause. Verbs take 는데; adjectives take -(으)ㄴ데; nouns take 인데.

한국 생활은 힘든데 재미있어요.

Life in Korea is hard, but it's fun. (힘들다 + ㄴ데, ㄹ drops → 힘든데)

지금 비가 오는데 우산이 없어요.

It's raining now, but I don't have an umbrella. (verb 오다 → 는데)

배가 고픈데 식당이 문을 닫았어요.

I'm hungry, but the restaurant is closed. (adjective 고프다 → ㄴ데)

이 옷이 예쁜데 좀 비싸요.

These clothes are pretty, but a bit expensive.

Compare 지만 is purely "but"; 는데 is softer — it often just sets the scene ("…, and so…") and is the natural lead-in to a request: 시간이 있는데 같이 갈까요? ("I have time, so shall we go together?").
