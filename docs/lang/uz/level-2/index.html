2-daraja · 초급 2

한국어와 한국문화 초급 2 (18 boʻlimli nashr) uchun mustaqil oʻrganish qoʻllanmasi · Koreya immigratsiya va integratsiya dasturi

Ushbu qoʻllanma KIIP 2-daraja darsligi (한국어와 한국문화 초급 2) — Ijtimoiy integratsiya dasturining boshlangʻich-yuqori bosqichi asosida tuzilgan. U darslikdagi 18 boʻlim tartibida joylashtirilgan. Har bir boʻlimda kundalik mavzu, oʻsha boʻlimda oʻrgatiladigan ikkita grammatik qolip (shakli, maʼnosi va misol gaplari bilan) hamda mavzuga mos lugʻat keltiriladi. Qoʻllanma soʻngida grammatika koʻrsatkichi beriladi. Bularning bari 1-darajada oʻrganilgan yuklama va asosiy feʼl qoʻshimchalari ustiga bevosita quriladi.

1-BOʻLIM고향 소개Tugʻilgan yurt bilan tanishtirish

N + (이)라고 하다 — "deb ataladi / nomlanadi"

Shakli undosh bilan tugagan N + 이라고 하다 · unli bilan tugagan N + 라고 하다

Biror narsaning nomini yoki uning nima deb atalishini aytish uchun ishlatiladi; oʻzini tanishtirishda ham qoʻllanadi. Undosh bilan tugagan ot + 이라고 하다, unli bilan tugagan ot + 라고 하다.

제 고향은 부산이라고 해요.

Mening tugʻilgan yurtim Pusan deb ataladi. (부산 ㄴ bilan tugaydi → 이라고)

저는 흐엉이라고 해요.

Mening ismim Xyong.

이 음식은 한국말로 김치찌개라고 해요.

Bu taom koreyscha kimchchige deb ataladi. (김치찌개 unli bilan tugaydi → 라고)

제 딸 이름은 민지라고 해요.

Qizimning ismi Minji.

A + (으)ㄴ + N — "otni aniqlovchi sifat"

Shakli undosh bilan tugagan sifat negizi + 은 · unli bilan tugagan sifat negizi + ㄴ, otdan oldin

Sifatni otdan oldin keladigan aniqlovchiga aylantiradi ("chiroyli joy"). Undosh negiz + 은, unli negiz + ㄴ.

경치가 아름다운 곳이에요.

Bu manzarasi goʻzal joy. (아름답다 — ㅂ tovushli notoʻgʻri feʼl → 아름다운)

조용한 동네에 살아요.

Men tinch mahallada yashayman. (조용하 unli bilan tugaydi → ㄴ)

우리 고향은 공기가 깨끗한 곳이라서 살기 좋아요.

Bizning yurtimiz havosi toza joy boʻlgani uchun yashashga qulay.

날씨가 좋은 날에는 공원에서 산책해요.

Havo chiroyli kunlari bogʻda sayr qilaman. (좋 undosh bilan tugaydi → 은)

Maslahat 있다/없다 sifatlari 은 emas, 는 oladi: 맛있는 음식, 재미없는 영화. ㅂ tovushli notoʻgʻri negizlarga ham eʼtibor bering: 춥다 → 추운 날, 맵다 → 매운 음식.

2-BOʻLIM집안일Uy ishlarini bajarish

V + (으)ㄹ게요 — "men ... qilaman (vaʼda / niyat)"

Shakli undosh negiz + 을게요 · unli negiz + ㄹ게요 (faqat birinchi shaxs)

Soʻzlovchining irodasini yoki vaziyatga, suhbatdoshga javoban bergan vaʼdasini bildiradi. Undosh negiz + 을게요, unli negiz + ㄹ게요.

쓰레기는 제가 버릴게요.

Axlatni men chiqaraman. (버리 unli bilan tugaydi → ㄹ게요)

제가 설거지를 할게요.

Idishlarni men yuvaman.

빨래는 제가 갤게요. 청소기는 남편이 돌려요.

Kirlarni men taxlayman. Changyutgich bilan esa erim tozalaydi.

밥 다 먹을게요. 조금만 기다려 주세요.

Ovqatimni yeb olaman. Birozgina kutib turing. (먹 ㄱ bilan tugaydi → 을게요)

Qiyoslang -(으)ㄹ게요 — suhbatdoshga beriladigan vaʼda (faqat birinchi shaxs); -(으)ㄹ 래요 esa oʻz xohishini bildiradi yoki suhbatdoshning xohishini bemalol soʻraydi. 가실게요 deyilmaydi — boshqa kishi nomidan vaʼda berib boʻlmaydi.

V + (으)ㄴ 다음에 — "...gandan keyin"

Shakli undosh negiz + 은 다음에 · unli negiz + ㄴ 다음에 (= 후에)

Ikkinchi ish birinchisi tugagach roʻy berishini bildiradi. Undosh negiz + 은 다음에, unli negiz + ㄴ 다음에.

청소한 다음에 쉴 거예요.

Tozalab boʻlgach, dam olaman. (청소하 unli bilan tugaydi → ㄴ 다음에)

밥을 먹은 다음에 약을 드세요.

Ovqatlangandan keyin dorini iching. (먹 ㄱ bilan tugaydi → 은 다음에)

손을 씻은 다음에 요리를 시작해요.

Qoʻlimni yuvgandan keyin ovqat tayyorlashga kirishaman.

빨래를 한 다음에 마트에 갈 거예요.

Kir yuvib boʻlgach, doʻkonga boraman.

Eslatma Bu yerda 다음에 va 후에 bir-birini almashtira oladi (먹은 다음에 = 먹은 후에). 후에 otga toʻgʻridan-toʻgʻri qoʻshiladi: 식사 후에 "ovqatdan keyin".

3-BOʻLIM옷, 신발Kiyim kiyib koʻrish

N + (이)나 — "yoki"

Shakli undosh bilan tugagan N + 이나 · unli bilan tugagan N + 나

Ikki otni "A yoki B" tarzida bogʻlaydi. Undosh bilan tugagan ot + 이나, unli bilan tugagan ot + 나. (Feʼllardan keyin -거나 ishlatiladi.)

치마나 바지를 입어요.

Yubka yoki shim kiyaman. (치마 unli bilan tugaydi → 나)

커피나 차를 마셔요.

Qahva yoki choy ichaman.

주말에는 운동화나 샌들을 신어요.

Dam olish kunlari krossovka yoki sandal kiyaman.

점심으로 빵이나 김밥을 먹을까 해요.

Tushlikka non yoki kimpap yemoqchiman. (빵 undosh bilan tugaydi → 이나)

V + 아 / 어 보다 — "...ib koʻrmoq"

Shakli ㅏ/ㅗ negiz + 아 보다 · boshqa negizlar + 어 보다

Biror ishni sinab koʻrishni bildiradi. ㅏ/ㅗ negiz + 아 보다, boshqa negizlar + 어 보다. Oʻtgan zamonda (아/어 봤다) "ilgari qilib koʻrgan" maʼnosini beradi.

이거 한번 입어 보세요.

Buni bir kiyib koʻring. (입 → 어 보다)

한국 음식을 먹어 봤어요.

Koreys taomini yeb koʻrganman. (oʻtgan zamon 봤다 = ilgari qilib koʻrgan)

이 신발이 편한지 한번 신어 보세요.

Bu poyabzal qulaymi-yoʻqmi, bir kiyib koʻring.

작년에 한복을 입어 봤는데 정말 예뻤어요.

Oʻtgan yili hanbok kiyib koʻrgandim, juda chiroyli ekan.

4-BOʻLIM전화Telefonda gaplashish

V + (으)ㄹ 수 있다 / 없다 — "...a olmoq / ...a olmaslik"

Qobiliyat yoki imkoniyatni (있다) hamda imkonsizlikni (없다) bildiradi. Undosh negiz + 을 수 있다/없다, unli negiz + ㄹ 수 있다/없다.

5-BOʻLIM약, 증상Dori va kasallik

A/V + (으)면 — "agar / ...sa"

Shakli undosh negiz + 으면 · unli negiz + 면

Shart bildiradi — "agar X (boʻlsa), u holda Y". Undosh negiz + 으면, unli negiz + 면.

많이 아프면 이 약을 드세요.

Qattiq ogʻrisa, shu dorini iching. (아프 unli bilan tugaydi → 면)

시간이 있으면 같이 가요.

Vaqtingiz boʻlsa, birga boraylik. (있 ㅆ bilan tugaydi → 으면)

열이 안 내리면 내일 다시 병원에 오세요.

Harorat tushmasa, ertaga yana shifoxonaga keling.

약을 먹으면 좀 괜찮아질 거예요.

Dorini ichsangiz, birmuncha yengil tortasiz. (먹 ㄱ bilan tugaydi → 으면)

V + 아 / 어서 — "...gani uchun / ...gach"

Shakli ㅏ/ㅗ negiz + 아서 · boshqa negizlar + 어서 · 하다 → 해서

Sababni natijaga, yoki ketma-ket bogʻliq ikki ishni bogʻlaydi. ㅏ/ㅗ negiz + 아서, boshqa negizlar + 어서, 하다 → 해서. Birinchi gapda zamon koʻrsatkichi qoʻyilmaydi.

감기에 걸려서 병원에 갔어요.

V + 는 게 어때요? — "...sangiz qanday boʻladi?"

Shakli feʼl negizi + 는 게 어때요? (는 게 = 는 것이)

Taklif yoki tavsiya berishning doʻstona usuli. Feʼl negizi + 는 게 어때요? (는 게 = 는 것이).

이 수업을 신청하는 게 어때요?

Shu darsga yozilsangiz qanday boʻladi?

택시를 타는 게 어때요?

Taksiga oʻtirsangiz qanday boʻladi?

시간이 없으면 인터넷으로 신청하는 게 어때요?

Vaqtingiz boʻlmasa, internet orqali ariza topshirsangiz qanday boʻladi?

주말에 요리 교실을 같이 들어 보는 게 어때요?

Dam olish kuni birga oshpazlik darsiga qatnashib koʻrsangiz qanday boʻladi?

A + 아 / 어 보이다 — "...koʻrinadi (tashqi koʻrinishidan)"

Shakli ㅏ/ㅗ sifat negizi + 아 보이다 · boshqa sifat negizlari + 어 보이다

Biror narsa tashqi koʻrinishiga qarab qanday koʻrinishini bildiradi. ㅏ/ㅗ sifat negizi + 아 보이다, boshqa sifat negizlari + 어 보이다.

그 옷이 따뜻해 보여요.

U kiyim issiq koʻrinadi. (따뜻하 → 해 보이다)

오늘 기분이 좋아 보여요.

Bugun kayfiyatingiz yaxshi koʻrinadi. (좋 ㅗ unlisi → 아 보이다)

이 요가 수업이 재미있어 보여요.

Bu yoga darsi qiziqarli koʻrinadi.

며칠 못 주무셨어요? 좀 피곤해 보여요.

Bir necha kun uxlamadingizmi? Biroz charchagan koʻrinasiz.

Qiyoslang -아/어 보이다 tashqi koʻrinish boʻyicha baho beradi ("charchagan koʻrinadi"); -(으)ㄴ/는 것 같다 esa har qanday belgidan kelib chiqqan kengroq taxmin. 아파 보여요 = kasaldek koʻrinasiz; 아픈 것 같아요 = kasal boʻlsangiz kerak.
