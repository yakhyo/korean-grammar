# KIIP Grammar &amp; Vocabulary

[English](../README.md) · [Oʻzbekcha](README.uz.md) · **Русский** · [Bahasa Indonesia](README.id.md)

Справочный комплект для самостоятельного изучения корейского языка по программе **«Иммиграция и интеграция в Корее» (KIIP / 사회통합프로그램)**.

🔗 **Действующий сайт:** https://yakhyo.github.io/korean-grammar/

## Содержание

| Уровень | Учебник | Темы |
|-------|------|-------|
| Уровень 1 | 한국어와 한국문화 초급 1 | Начальный — хангыль, основные частицы, базовые окончания глаголов |
| Уровень 2 | 한국어와 한국문화 초급 2 | Выше начального — союзы, условия, относительные обороты |
| Уровень 3 | 한국어와 한국문화 중급 1 | Средний — косвенная речь, предположение, приложение со спряжениями |
| Уровень 4 | 한국어와 한국문화 중급 2 | Выше среднего — общественные темы, абстрактная грамматика |
| Уровень 5 | 한국사회 이해 (기본·심화) | Понимание корейского общества — учебное пособие по обществознанию |

## Возможности

- 🌐 **Переключатель языка** — выпадающее меню на каждой странице для переключения между английским, узбекским, русским и индонезийским; корейский текст остаётся, меняются только пояснения и переводы.
- 🧭 **Переключатель уровней** — выпадающее меню на каждой странице, позволяющее сразу перейти к любому уровню (или на главную) без возврата назад.
- 🌗 **Тёмная / светлая тема** — переключатель на каждой странице; выбор запоминается между визитами и по умолчанию следует настройкам вашей системы.
- ⭐ **Кнопка «Звезда»** — быстрая ссылка, чтобы отметить проект звездой на GitHub.
- 🖨️ **Готово к печати** — при печати (или сохранении в PDF) всегда используется светлая тема, чтобы результат выглядел аккуратно — даже в тёмном режиме.
- 📦 **Без сборки и зависимостей** — простой HTML + один общий файл CSS, работает офлайн.

## О проекте

- Оформление хранится в двух общих таблицах стилей (`assets/home.css` — для главных страниц, `assets/levels.css` — для всех страниц уровней); каждая страница уровня выбирает свой акцентный цвет через класс `html.lv1…lv5`. Страницы по-прежнему работают офлайн и аккуратно печатаются в PDF (`Ctrl`/`Cmd` + `P`).
- Уровни 1–4 соответствуют текущим изданиям учебников KIIP; списки лексики — это расширенные тематические справочники, а не дословные словарные списки уроков.
- К каждому грамматическому пункту прилагаются **четыре разобранных примера предложений**, показывающих закономерность в разных контекстах.
- Уровень 5 охватывает курс *Понимание корейского общества*.
- Узбекская, русская и индонезийская версии — это полные переводы пояснений, примеров и лексики; сам корейский текст одинаков во всех четырёх языках.

## Структура проекта

```
.
├── docs/                       # Опубликованный сайт — GitHub Pages раздаёт эту папку
│   ├── index.html              # Перенаправление → /lang/en/ (корень сайта)
│   ├── lang/                   # По одной папке на язык
│   │   ├── en/                 # Английский — index.html (→ /lang/en/) + level-1/ … level-5/
│   │   ├── uz/                 # Узбекский (→ /lang/uz/, та же структура)
│   │   ├── ru/                 # Русский
│   │   └── id/                 # Индонезийский
│   ├── assets/                 # Общие таблицы стилей и превью для соцсетей
│   │   ├── home.css            # Главные страницы
│   │   ├── levels.css          # Страницы уровней (акцент через класс html.lvN)
│   │   └── og-image*.png       # Превью-изображения для соцсетей по языкам
│   ├── sitemap.xml             # Список URL с альтернативами hreflang
│   └── .nojekyll               # Раздача файлов как есть на GitHub Pages
├── tools/                      # Служебные скрипты (без зависимостей)
│   └── check-korean-sync.mjs   # Проверка совпадения корейского в EN/UZ/RU/ID
├── translations/               # Переводы README (uz · ru · id)
└── README.md                   # Этот файл (английский)
```

## Запуск локально

Этап сборки не нужен — запустите локальный сервер для папки `docs/`:

```bash
python3 -m http.server 8000 --directory docs
# затем откройте http://localhost:8000
```
